## 🚀 Comandos para Publicar

Ahora ejecuta esto en PowerShell:

```powershell
# 1. Ver qué se va a subir (ya ejecutado)
git status

# 2. Crear commit
git commit -m "feat: Calculadora web para amigurumis con Supabase y OAuth"

# 3. Configurar remote (si no existe)
git remote add origin https://github.com/andresospinacg/arte-entre-puntadas.git

# 4. Cambiar a rama main
git branch -M main

# 5. Subir a GitHub
git push -u origin main
```

## 🔄 Si necesitas modificar el README

El `README.md` SÍ se subirá a GitHub, así que:

1. Edítalo con información pública del proyecto
2. Incluye instrucciones de instalación
3. NO incluyas keys secretas o información sensible
4. Ejemplo de contenido apropiado:

```markdown
# Arte Entre Puntadas - Calculadora de Amigurumis

Calculadora web profesional para calcular precios de proyectos de amigurumis.

## Características

- 🧮 Cálculo automático de costos
- 📦 Gestión de inventario de materiales
- 📊 Reportes y estadísticas
- 🔐 Autenticación con Google OAuth
- ☁️ Sincronización en la nube con Supabase

## Instalación

1. Clonar el repositorio
2. Instalar dependencias: `npm install`
3. Configurar variables de entorno (ver `.env.example`)
4. Ejecutar: `npm run dev`

## Tecnologías

- Astro
- React
- TypeScript
- Tailwind CSS
- Supabase
```
